# Laff-A-Lot Fundraisers One-Page Site

## Files
- `index.html`
- `styles.css`
- `script.js`
- `assets/logo.png`

## Run locally
1. Open `index.html` directly in your browser.
2. Optional: from this folder run `python3 -m http.server 8080` and visit `http://localhost:8080`.

## Notes
- Contact email is set to `ltmullaney24@gmail.com` in `index.html` and `script.js`.
- The primary CTA and contact form both launch your default mail client.
